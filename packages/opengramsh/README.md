# opengramsh

This is a convenience wrapper that lets you install [**@opengramsh/opengram**](https://www.npmjs.com/package/@opengramsh/opengram) without the scoped name:

```bash
npm install -g opengramsh
# instead of
npm install -g @opengramsh/opengram
```

Both packages provide the same `opengram` CLI command. For documentation, see the [main repository](https://github.com/opengramsh/opengram).
